# 자료구조및알고리즘 과제3

신규 아이디 추천 프로그램 생성
- https://programmers.co.kr/learn/courses/30/lessons/72410

조건
- 자료구조로 연결 리스트를 사용해야 합니다.
- 입력된 문자열을 사용자에게서 받아, 각 문자를 연결리스트의 하나의 노드에 저장합니다.
- 수업시간에 학습한 연결 리스트 ADT(라이브러리)를 사용해야 합니다. (단, 포인터 기반 연결 리스트만 사용가능)
- 만약 수업시간에 학습한 연결 리스트에 변경을 원할 경우, 무엇이 어떻게 변경되었는지와 이유를 설명해야 합니다.
- 위의 문제에 있는 함수 이외의 함수를 정의해도 됩니다.
